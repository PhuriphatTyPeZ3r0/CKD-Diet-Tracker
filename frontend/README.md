# CKD Diet Tracker

โปรเจกต์ Next.js สำหรับค้นหาอาหารที่เหมาะสมกับผู้ป่วยโรคไตเรื้อรังระยะที่ 3 และบันทึกการกินรายวัน

## เริ่มต้นใช้งาน

```bash
cmd /c npm install
cmd /c npm run dev
```

จากนั้นเปิด `http://localhost:3000`

## โครงสร้างหลัก

- `app/page.js` หน้าแรกของแอป
- `components/CKDDietApp.jsx` คอมโพเนนต์หลักจากโค้ดที่ให้มา
- `app/globals.css` Tailwind CSS globals
